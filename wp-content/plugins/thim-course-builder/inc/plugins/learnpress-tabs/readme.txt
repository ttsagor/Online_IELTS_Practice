== Description ==  
LearnPress Tabs is an extension plugin to provide tabs for course in LearnPress.

== Installation ==

**From your WordPress dashboard**  
1. Visit 'Plugin > Add new'.  
2. Search for 'LearnPress Tabs'.
3. Activate LearnPress Tabs from your Plugins page.

**From WordPress.org**  
1. Search, select and download LearnPress Tabs.
2. Activate the plugin through the 'Plugins' menu in WordPress Dashboard.
