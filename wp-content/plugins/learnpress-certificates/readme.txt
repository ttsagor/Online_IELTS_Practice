== Description ==
Create certificates for courses of LearnPress.

== Installation ==

**From your WordPress dashboard**  
1. Visit 'Plugin > Add new'.  
2. Search for 'LearnPress Certificates'.
3. Activate LearnPress from your Plugins page.
